Operadores Parte 2

Nessa parte, vamos tratar dos operadores, Relacionais, Lógicos e Ternário

Operadores Relacionais

Importante: Para todo expressão que contém operador relacional ligado a ela, o resultado sempre vai ser um valor booleano, ou seja, true ou false.

  • 5 < 2 -> true (verdadeiro)
  • 7 > 4 -> false (falso)
  • 8 >= 8 -> true (verdadeiro)
  • 9 <= 7 -> false (falso)
  • 5 == 5 -> true (verdadeiro)
  • 4 != 4 -> false (falso)

Exemplos:

  • preço >= 200.50
  • idade < 18
  • curso == 'JavaScript'
  • n1 != n2

Operadores de Identidade

Para o JavaScrip, o sinal de igualdade(==) não testa o tipo, justamento por isso, que no segundo exemplo abaixo, ele diz que 5(Number) é igual a '5'(String)

5 == 5 -> true 5 == '5' -> true 5 === '5' -> false

Operadores Lógicos

Os operadores lógicos são o de negação(!), conjunção(&&) e disjunção(||)

  • ! negação equivalende a: NÃO (NOT)
  • && conjunção equivalente a: E (AND)
  • || disjunção equivalende a: OU (OR)

Operador de Negação

O operador de negação, NOT, recebe apenas dois valores, true ou false.

  • !true -> uma coisa que NÃO é verdadeira é -> false
  • !false -> uma coisa que NÃO é falsa é -> true

Operador de Conjunção

O operador de conjunção é binário, e recebe dois valores lógicos de cada lado, exemplo:

  • true && true -> true
  • true && false -> false
  • false && true -> false
  • false && false -> false

Observação: *Só me satisfaz se as duas condições lógicas forem verdadeiras

Operador de disjunção

basta que um dos valores lógicos seja verdadeiro para ele ser veradeiro também.

  • true || true -> true
  • true || false -> true
  • false || true -> true
  • false || false -> false

IMPORTANTE: Caso haja os 3 operadores na mesma expressão, a ordem de execução é a segunte: Primeiro o NÃO, depois o E e por final o OU.

Exemplos

idade >= 15 && idade <= 17 estado == 'RJ' || estado == 'SP' salário > 1500 && sexo != 'M'

Ordem de Precedência dos Operadores

Dentro de uma expressão, vamos iniciar pelos:

  • Operadores Aritméticos -> (), **, /, -, +
  • Operadores relacionais -> >, <, >=, <=, !=
  • Operadores de Precedência -> !, &&, ||

Operador Ternário

O operador ternário é composto por dois símbolos, interrogação (?) e dois pontos (:). Se chama ternário porque é compostos por 3 partes:

teste ? true : false

A primiera parte é o testo lógico, que gera uma responsta verdadeiro ou falso.

A segunda parte é o que acontecer se o teste lógico for VERDADEIRO.

E a terceira parte é o que vai acontecer se o teste lógico for FALSO.

Exemplo:

média >= 7.0 ? 'APROVADO' : 'REPROVADO'

Se a médio for maior ou igual a 7.0, então está APROVADO, se for menor que 7.0, está REPROVADO!

Exemplo 2:

var idade = 19 var resp = idade >= 18 ? 'MAIOR' : 'MENOR'

Se a idade foi maior ou igual a 18, retorna MAIOR, Se não, retorna MENOR
