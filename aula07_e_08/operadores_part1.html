Ordem de Precedencia

A order de precedencia é extremamande importante, então é bom estar muito atento a ela quando for fazer cálculos.

  • () --> tudo o que tiver dentro do parentese primeiro;
  • ** --> Potência em segundo;
  • *, /, % --> Multiplicação, divisão e resto da divisão em terceiro;
  • +, - --> Adicão e subtração em quarto.

Atribuição Simples

    |
  • var a = 5 + 3 | 8
  • var b = a % 5 | 3
  • var c = 5 * b ** 2 | 45
  • var d = 10 - a / 2 | 6
  • var e = 6 * 2 / d | 2
  • var f = b % e + 4 / e | 3

AUTO-ATRIBUIÇÕES

  • var n = 3
  • n = n + 4
  • n = n - 5
  • n = n * 4
  • n = n / 2
  • n = n ** 2
  • n = n % 5

Simplificando Atribuições

  • n += 4
  • n -= 5
  • n *=4
  • n /= 2
  • n **= 2
  • n %= 5

Atribuição por INCREMENTO

Além das atribuições acima também há a por INCREMENTO, exeplo abaixo:

x = x + 1, pode ser assim => x++

x = x -1 , pode ser assim => x--

Observação: o ++ e o --, podem também vir antes do x. isso quer fizer que ele vai somar ou subtrair antes de armazenar e não armazenar para depois fazer o calculo e mostrar o valor certo na tela. Na prática, em coistos básico e intermediários não faz tanta diferenças, mas na programação avançada ajuda bastante.
